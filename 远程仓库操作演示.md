
# GitHub 远程仓库操作演示

## 本次操作流程记录

### ✅ 已完成的步骤
1. 在 GitHub 上创建了空仓库：https://github.com/GXiaokang/git-learning-project.git
2. 在本地添加远程仓库：`git remote add origin https://github.com/GXiaokang/git-learning-project.git`
3. 将主分支重命名为 main：`git branch -M main`
4. 首次推送到远程仓库：`git push -u origin main`

### 📋 后续常用操作

#### 修改文件后推送到 GitHub
```bash
# 1. 修改文件（比如修改 README.md）
# 2. 查看状态
git status

# 3. 添加到暂存区
git add .

# 4. 提交
git commit -m "提交信息"

# 5. 推送到 GitHub
git push origin main
```

#### 从 GitHub 拉取最新代码
```bash
git pull origin main
```

## 记住这些关键命令

|------|------|
